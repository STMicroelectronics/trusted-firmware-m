M33TDCID
========

In M33 TDCID mode, the cortex M33 is the owner of Trusted Domain (TDCID), is in charge of:

* security setup (memories, peripherals) for all platform (cortex A, M)
* clock tree setting.
* start cortex A.

This document contains specific topic for M33 TDCID in stm32mp2 platform.
